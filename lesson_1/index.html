XHTML, acronimo en inglés de eXtensible Hypertext Markup Language
(lenguaje extensible de marcado de hipertexto), es el lenguaje de marcado
pensado para sustituir a HTML como estandar para las páginas Web. En su
version 1.0, XHTML es solamente la version XML de HTML, por lo que tiene,
básicamente, las mismas funcionalidades, pero cumple las especificaciones,
más estrictas de XML.
